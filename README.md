gulp-append-ver
====
#安装
1、运行node-v5.7.0-x64.msi文件，安装nodejs
2、运行install.bat下载gulp，国内需翻墙
3、gulp安装完成后，目录下会多出一个node_modules文件夹，将原有node_modules_modify文件夹内容拷贝进去覆盖
